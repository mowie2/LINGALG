Om het spel te spelen ga start de exe op die op de volgende locatie staat:
LINGALG\LineareAlgebra\x64\Debug\LineareAlgebra.exe

De sourcecode staat in:
LINGALG\LineareAlgebra\LineareAlgebra

controls:
Q 	= barrelroll tegen de klok in
E 	= barrelroll met de klok mee
D 	= naar rechts draaien
A 	= naar links draaien
W 	= omhoog 
E 	= omlaag
Spatie 	= naar voren gaan
C 	= stoppen
Y 	= schieten

Camera kan je roteren met de pijltjes toetsen
